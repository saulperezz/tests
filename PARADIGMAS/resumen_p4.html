Práctica 4: Recursión con Listas

LPP · Práctica 4·5 conceptos clave
1
Función es-prefijo?
Concepto
Verificar si una lista es prefijo de otra (todos sus elementos aparecen al inicio en el mismo orden).
(es-prefijo? '(a b) '(a b c d)) ; => #t (es-prefijo? '() '(a b c)) ; => #t
2
Insertion Sort
Concepto
Ordenar una lista insertando cada elemento en su posición correcta. Las funciones son inserta-ordenada e ordena.
(inserta-ordenada 5 '(1 3 7 9)) ; => (1 3 5 7 9) (ordena '(3 1 4 1 5)) ; => (1 1 3 4 5)
3
Función mueve-al-principio
Concepto
Extraer un elemento específico de una lista y ponerlo al principio.
(mueve-al-principio 'b '(a b c d)) ; => (b a c d)
4
Función expande
Concepto
Expandir una lista de parejas, donde cada pareja (n . x) se convierte en n copias de x.
(expande '((2 . a) (3 . b))) ; => (a a b b b)
5
Lambdas y HOF
Concepto
Funciones anónimas usando lambda, composición con map, y funciones de orden superior.
((lambda (x y) (* x (+ 2 y))) (+ 2 3) 4) ; => 30 (map (lambda (x) (* x 2)) '(1 2 3)) ; => (2 4 6)
