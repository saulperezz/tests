Práctica 3: Recursión

LPP · Práctica 3·6 conceptos clave
1
Función minimo
Concepto
Encontrar el mínimo de una lista comparando recursivamente el primer elemento con el mínimo del resto.
Expresiones clave
(minimo '(1 8 6 4 3)) ; => 1 (minimo '(1 -1 3 -6 4)) ; => -6
2
Función concatena
Concepto
Convertir una lista de caracteres en una cadena usando recursión y string-append.
Expresiones clave
(concatena '(#\H #\o #\l #\a)) ; => "Hola" (concatena '()) ; => ""
3
Función contiene?
Concepto
Buscar un elemento en una lista devolviendo #t si existe, #f si no. Dos versiones: contiene? y str-contiene?.
Expresiones clave
(contiene? '(algo 3 #\A) 3) ; => #t (str-contiene? "Hola" #\o) ; => #t (contiene? '() 5) ; => #f
4
Función todos-iguales?
Concepto
Verificar si todos los elementos de una lista son iguales entre sí.
Expresiones clave
(todos-iguales? '(a a a a)) ; => #t (todos-iguales? '(a a a b)) ; => #f
5
Función todos-distintos?
Concepto
Verificar que todos los elementos de una lista son distintos entre sí (sin repetidos).
Expresiones clave
(todos-distintos? '(a b c)) ; => #t (todos-distintos? '(a b c a)) ; => #f
6
Análisis de Recursión
Concepto
Entender cómo se estructuran las llamadas recursivas: caso base, parámetro que se reduce, y composición del resultado.
Explicación
En (minimo '(1 8 6 4 3)):
  • Primera llamada recursiva recibe: '(8 6 4 3)
  • Esa llamada devuelve: 3 (el mínimo del resto)
  • Composición final: (menor 1 3) devuelve 1
