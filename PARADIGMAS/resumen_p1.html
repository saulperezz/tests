1
Expresiones Numéricas
Concepto
En Scheme, los números se pueden combinar con operadores aritméticos usando notación prefija: (+ 1 2) significa 1 + 2.
Explicación
La notación prefija sitúa el operador antes de los operandos. Esto es diferente a la notación infiija (1 + 2) que usamos en matemáticas. Los operadores comunes son:
  • + suma números: (+ 1 2 3) = 6
  • - resta o niega: (- 10 3) = 7
  • * multiplica: (* 2 3 4) = 24
  • / divide: (/ 10 2) = 5
  • sqrt raíz cuadrada: (sqrt 25) = 5
  • expt potencia: (expt 2 3) = 8
Los operadores pueden combinarse de forma anidada para expresiones complejas.
Expresiones clave
(+ 1 2 3 4) ; => 10 (* (+ 2 3) 5) ; => 25 (not (> (+ 3 2) 5)) ; => #t (and (even? 2) (odd? 5)) ; => #t
2
Parejas
Concepto
Una pareja es una estructura que contiene dos elementos: (cons a b) crea una pareja con componente izquierdo a y derecho b.
Explicación
Las parejas son fundamentales en Scheme. Para crearlas y acceder a ellas usamos:
  • cons crea una pareja: (cons 1 2) → '(1 . 2)
  • car obtiene el componente izquierdo: (car (cons 1 2)) = 1
  • cdr obtiene el componente derecho: (cdr (cons 1 2)) = 2
Las parejas se pueden anidar: (cons (cons 1 2) 3) crea una pareja cuyo componente izquierdo es otra pareja. Son la base de los árboles y las listas.
Expresiones clave
(car (cons 1 2)) ; => 1 (cdr (cons 1 2)) ; => 2 (car (cons (cons 3 4) 2)) ; => '(3 . 4) (cdr (cons 1 (cons 2 3))) ; => '(2 . 3)
3
Listas
Concepto
Una lista es una secuencia de elementos encadenados mediante parejas. (list 1 2 3) es sintaxis azucarada para (cons 1 (cons 2 (cons 3 '()))).
Explicación
Las listas son el tipo de dato más importante en lenguajes funcionales. Para trabajar con ellas:
  • list crea una lista: (list 1 2 3)
  • first (o car) obtiene el primer elemento
  • rest (o cdr) obtiene el resto de la lista sin el primero
  • cons agrega un elemento al frente: (cons 5 (list 1 2)) = '(5 1 2)
El símbol '() representa la lista vacía. Internamente, '(1 2 3) es (cons 1 (cons 2 (cons 3 '()))).
Expresiones clave
(first (rest (rest (list 1 2 3 4)))) ; => 3 (cons 3 '(1 2 3)) ; => '(3 1 2 3) (first (list (list 1 2) 3)) ; => '(1 2) (first (rest (rest (rest '(1 2 3 4))))) ; => 4
4
Strings y Caracteres
Concepto
Un string es una secuencia de caracteres entre comillas dobles. Un carácter se escribe como #\a, donde el símbolo representa el carácter específico.
Explicación
Scheme distingue entre strings (secuencias) y caracteres (unidades individuales). Las funciones principales son:
  • string-ref accede a un carácter por índice (comenzando en 0): (string-ref "hola" 1) = #\o
  • substring extrae una porción: (substring "buenos" 1 4) = "uen"
  • string-length devuelve la longitud: (string-length "hola") = 4
  • equal? compara strings: (equal? "hola" "hola") = #t
Los índices siempre comienzan en 0, y en substring el segundo índice es exclusivo.
Expresiones clave
(string-ref "pepe" 1) ; => #\e (substring "buenos dias" 1 4) ; => "uen" (equal? "hola" "hola") ; => #t (string-length "ejemplo") ; => 7
